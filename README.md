# 🌙 Serenity — Mental Wellness Web App

A safe, beautiful space for people dealing with loneliness, anxiety, and depression.

Serenity is a fully client-side mental wellness web application built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies, no build tools. Just one HTML file you can open in any browser.

Designed for people experiencing loneliness, anxiety, depression, or everyday stress, Serenity provides a calm midnight-purple space to express feelings, track moods, breathe mindfully, meditate, and chat with an AI companion called Sage.

## Features

- 🧠 Mood Check-in — 8 emotion cards with personalised supportive responses
- 📊 Mood Tracker Chart — 7-day bar chart of logged moods
- 📓 Private Journal — write freely with guided prompts, edit & delete entries
- 🌬️ Box Breathing — 4-phase guided breathing with animated rings
- 🧘 Guided Meditation — 4 sessions (3 / 5 / 10 / 15 min) with progress ring
- 😴 Sleep & Gratitude Log — rate your sleep and record 3 daily gratitudes
- ✅ Self-care Checklist — 8 daily habits, auto-resets each day
- 🏅 Progress Badges — 12 milestones earned automatically as you use the app
- 💬 Motivational Quotes — 7 healing quotes with auto-rotation
- 🌅 Daily Affirmation — a new affirmation tied to today's date
- 🎵 Mood Relaxing Music — 9 curated tracks (Calm, Sleep, Anxiety Relief, Focus, Nature)
- 🤖 AI Companion Sage — empathetic Claude-powered chat (requires Anthropic API key)
- ☰ Mobile-friendly — hamburger nav, responsive layout, fixed AI chat button

## Tech Stack

- Pure HTML5, CSS3, Vanilla JavaScript
- No frameworks, no npm, no build step
- localStorage for all data (private, stays on your device)
- Anthropic Claude API for the Sage AI companion
- YouTube nocookie embeds for music player
- Google Fonts (Cormorant Garamond + DM Sans)
- SEO optimised with structured data, Open Graph, and Twitter Card tags

## Getting Started

1. Clone or download the repository
2. Open index.html in any modern browser
3. To use the AI companion, get a free API key from console.anthropic.com and enter it in the Sage chat panel

## Privacy

All journal entries, mood logs, sleep history, gratitude logs, and checklist data are stored entirely in your browser's localStorage. Nothing is sent to any server except your messages to the Sage AI (which go directly to Anthropic's API using your own key).
